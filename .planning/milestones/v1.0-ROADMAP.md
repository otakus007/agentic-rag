# Milestone v1.0: Agentic RAG Platform — Archived Roadmap

**Completed:** 2026-03-21
**Stats:** 6 phases | 6 plans | 41 tests | 24 files | 1,190 LOC

## Key Accomplishments

1. **Core Scaffold** — FastAPI + LangGraph state machine with PostgresSaver checkpointing and ChatOpenAI integration.
2. **Semantic Memory** — Mem0 cross-session user memory with per-user/agent isolation and LLM-triggered `save_memory` tool.
3. **Document Ingestion** — MinerU PDF parsing → OpenAI text-embedding-3-small → Qdrant vector storage with per-agent collections (`kb_{agent_id}`).
4. **Full RAG Flow** — Dual context injection (Mem0 + Qdrant) with inline `[1]` citation markers and `/chat` endpoint returning structured `{answer, sources}`.
5. **OAuth & API Gateways** — Google/Microsoft SSO via JWT Bearer tokens with `user_id` derived from `sub` claim. `/health` public, `/chat` + `/ingest` protected.
6. **Custom Tool Extensions** — Per-`agent_id` dynamic tool registry with `build_graph()` compilation. Built-in `visualize` tool returning structured JSON (`chart`/`table`/`mermaid`).

## Phase Details

### Phase 1: Core Scaffold & LangGraph Hello World
**Goal:** Setup Poetry, FastAPI, basic LiteLLM integration, and a "Hello World" LangGraph state machine with memory persistence (PostgresSaver).

### Phase 2: Mem0 Integration & User Memory
**Goal:** Connect Mem0 for retrieving and storing cross-session semantic facts per `user_id`, independent of the short-term checkpoint logic.

### Phase 3: Document Ingestion Pipeline
**Goal:** Integrate MinerU for parsing PDFs and saving embeddings into Qdrant vector database.

### Phase 4: Full Multi-Tenant RAG Flow
**Goal:** Connect LangGraph to query Qdrant based on Mem0 injected history. Output proper citations.

### Phase 5: OAuth & API Gateways
**Goal:** Add Microsoft/Google SSO middleware to FastAPI. Associate `user_id` tokens with API requests to enforce multi-tenancy.

### Phase 6: Custom Tool/Visualizer Extensions
**Goal:** Introduce Tool Nodes safely isolated via conditional edges for executing action plans or generating UI graphs.
